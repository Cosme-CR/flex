# flex# Layout com Flexbox – HTML e CSS

Este projeto utiliza HTML5 e CSS3 para demonstrar o uso do Flexbox na organização de elementos na tela.

## Estrutura do Projeto

/
├── index.html
└── style.css

## Descrição

A página é composta por um elemento main que ocupa 100% da altura da tela (100vh).  
Dentro dele existem cinco seções horizontais, cada uma com 20% da altura total.

Cada seção contém quatro círculos coloridos e utiliza um tipo diferente de alinhamento com Flexbox.

## Seções

.um  
Alinhamento dos elementos à direita e na parte inferior.

.dois  
Elementos alinhados à direita e centralizados verticalmente.

.tres  
Elementos centralizados horizontal e verticalmente.

.quatro  
Elementos distribuídos com espaço entre eles.

.cinco  
Elementos distribuídos com espaço igual ao redor.

## Estilização

Os círculos são criados utilizando border-radius: 50% e bordas largas para formar o formato circular.

## Objetivo

Praticar:
- Flexbox
- Alinhamento de elementos
- Organização de layout
- Uso de cores e espaçamento

## Tecnologias Utilizadas

- HTML5
- CSS3
